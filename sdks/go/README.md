# gbz185-sdk Go

Go client SDK for GB/Z 185-2026 agent interconnection JSON operations.

```bash
go get github.com/IchenDEV/gbz185-sdk/sdks/go/gbz185
```

Local verification:

```bash
go test ./...
```

The Go SDK exposes the same JSON operation names as the TypeScript SDK and supports `InProcessJsonTransport` plus `HttpJsonTransport`.
