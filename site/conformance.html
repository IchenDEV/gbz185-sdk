边界说明

Covered 表示 SDK 类型或公开 API 已直接覆盖；Reference runtime 表示内存运行时已提供可运行参考实现；Extension point 表示国标涉及治理、基础设施、生产安全或界面形态，SDK 提供接口，不假装替你完成合规落地。

  • 生产级 CA、国密算法、主管部门证书策略：实现 CredentialIssuer、CredentialVerifier、CertificateChainVerifier。
  • 监管认可的身份注册治理：实现 IdentityAccountStore 并把审核流程接入 IdentityRegistryRuntime。
  • GUI/LUI 发现界面：基于 AgentInterconnectClient.discover 构建应用界面。
  • REST、WebSocket、MCP、消息队列：实现 JsonTransport 或在服务端适配 operation。
