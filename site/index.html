SDK 覆盖哪些运行链路

链路 关键 API 开发者拿它做什么
身份码 formatIdentityCode, parseIdentityCode, validateIdentityCode 生成、解析和校验 GB/Z 185.2 结构的智能体身份码。
身份与凭证 IdentityRegistryRuntime, CredentialIssuer, CredentialVerifier 注册身份账户、发行开发凭证、验证过程凭证包、处理锁定和注销。
描述与发现 AgentDescriptionRegistry, DiscoveryService 发布智能体描述，并按名称、自然语言、技能、标签、IO 类型和可用性查找。
交互与消息 InteractionRuntime, MessageDistributionRuntime 创建点对点、群组、混合会话，提交任务，发送分块消息和最终结果。
工具调用 ToolRuntime, ToolAccessRuntime 注册工具、同步工具列表、批量调用工具，并返回成功或失败状态。
传输适配 JsonTransport, InProcessJsonTransport, HttpJsonTransport 用统一 JSON operation 连接本地运行时、HTTP 服务或未来 MCP/WebSocket 适配器。

适合首版集成的场景

  • 在应用内验证 GB/Z 185 智能体互联流程。
  • 给现有智能体补身份码、描述、发现和工具调用表面。
  • 用内存运行时写集成测试或协议实验。
  • 用 JsonTransport 对接自己的 HTTP、WebSocket、消息队列或网关。

明确边界

  • 内置凭证实现是开发参考，不宣称生产 CA 或国密合规。
  • 存储默认是内存实现，生产环境应替换为数据库或状态服务。
  • SDK 不规定 REST 路径、MCP 方法名或 WebSocket 消息格式。
  • GUI/LUI 是应用层能力，SDK 提供可调用的客户端和模型。
